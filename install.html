Installing the CLI

To use it, pull this repo and build it with mvn clean install

This will build:

  • A Debian installer at target/unbound_1.0.0-1_all.deb
  • A RPM installer at target/rpm/unbound/RPMS/noarch/unbound-1.0.0-1.noarch.rpm
  • A Windows installer at target/unbound-1.0.0.exe

Install the appropiate one for your OS and/or distribution of Linux

For OSX use the RPM installer; you might have to install rpm via brew

What the installer does

It installs two files, unbound-1.0.0-exec.jar and either mvnu or mvnu.bat A copy of the Apache 2.0 License might also be included
